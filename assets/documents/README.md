# Resume and CV Documents

This directory contains the downloadable resume and CV files for the portfolio website.

## Required Files

Please add the following PDF files to this directory:

1. **Kristoffer_Kelly_Resume.pdf** - Your professional resume
2. **Kristoffer_Kelly_CV.pdf** - Your detailed curriculum vitae

## File Guidelines

- **Format**: PDF only
- **File Size**: Recommended under 2MB for web performance
- **Naming**: Use exact filenames as specified above
- **Content**: Ensure all personal information is current and accurate

## Portfolio Integration

These files are automatically linked in the portfolio's Resume section:
- Resume download button links to `./assets/documents/Kristoffer_Kelly_Resume.pdf`
- CV download button links to `./assets/documents/Kristoffer_Kelly_CV.pdf`

## Security Note

- These files will be publicly accessible via the portfolio website
- Ensure no sensitive information is included that you don't want public
- Consider using a professional email address and phone number
- Remove any personal addresses if privacy is a concern

## Updating Documents

1. Replace the existing PDF files with updated versions
2. Keep the same filenames for automatic linking
3. Test the download links after updating
4. Consider updating the "Updated [Date]" text in the portfolio if needed
